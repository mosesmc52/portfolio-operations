# portfolio-operations
App used to track quant portfolio

## Local development

#### System dependencies
* Install python 3.12.*
* Poetry


__**Step 1: Copy the env file from the backend folder**__
```
    cp .env.example .env
```

### System dependencies
 * Install docker
 * Install docker-compose


### Setup using docker-compose to run locally

__**Step 1: Copy the env file from the backend folder**__
```
    cp .env.example .env
```
